## 三网直连是什么？先把这个概念说清楚

**香港 VPS 三网直连，指的是国内三大运营商——中国电信、中国联通、中国移动——分别通过各自专用线路与香港机房直接连通，不绕路，不经第三国转发。**

这个定义说起来简单，但实际上"直连"质量差很多：

- 最顶的是**电信 CN2 GIA**：走的是电信自家的 CTG/GIA 国际精品网，双向都走，延迟低且稳定
- 次一级是**联通 AS9929 / AS10099**：联通自家精品线，效果不错
- **移动 CMI**：中国移动的国际骨干网，移动用户体验好

问题是，不同的 VPS 套餐，三网里每一张给你走什么线路，差别可以很大。这正是选香港 VPS 时最容易踩的坑。

## 常见问题

**Q：香港 VPS 三网直连是不是就一定用 CN2 GIA？**

不一定。CN2 GIA 是电信的精品线，三网直连只是说三家各有路可以直达香港。移动用 CMI、联通用 AS9929 也属于直连，效果各有不同。选哪个看你自己用的哪家运营商居多。

**Q：DMIT Premium 系列这么贵，有没有办法便宜一点？**

可以关注 DMIT 官方的促销活动，比如 Victoria 特惠套餐，年付 $298.88 折合每个月不到 $25，配置比直接月付 TINY 好很多。这类套餐有优惠码可以叠加，下单时在购物车输入框核对一下当前能用的活动码。

**Q：我是移动用户，Eyeball 和 Premium 哪个更适合？**

移动用户用 Eyeball 的体验已经不错了，因为两个系列移动这条线都是 CMI 直连。想进一步省钱的话 Eyeball 是合理选择。如果同时还有电信宽带或者对稳定性要求高，才有必要上 Premium。

**Q：Tier 1 可以用来翻墙、做代理吗？**

Tier 1 本身是服务器，技术上是可以装各种程序的。但它没有国内优化，从国内连过去延迟偏高，用来做回国节点的效果很差。真要走国内线路，看 Eyeball 或 Premium。

**Q：香港 VPS 对比洛杉矶有什么优势？**

物理距离。香港到大陆主要城市延迟普遍在 20~50ms 左右，洛杉矶一般在 150~200ms。如果你的用户在国内，或者你自己是国内用户用来建站，香港节点的体验差距是实实在在的。代价是价格比洛杉矶高一截。
